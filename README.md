# adivina-el-personaje-de-linux
🎭 La Máscara de Linux es un juego de adivinanza donde debes descubrir qué personaje del mundo Linux se esconde detrás de varias pistas. Cada ronda revela información mientras el tiempo corre. Gana puntos, crea combos y desbloquea logros con ayuda de Tux. Compite con otros jugadores y demuestra cuánto sabes del universo Linux.
